# Dynamical Mass Calculation using Python

This project calculates the dynamical mass of astronomical systems using astrophysical equations implemented in Python.

## Description
The notebook demonstrates how observational parameters can be used to compute the dynamical mass of celestial systems.

## Technologies Used
- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## Learning Outcome
- Applied Python for astrophysical calculations
- Implemented scientific equations programmatically
- Visualised astronomical data
